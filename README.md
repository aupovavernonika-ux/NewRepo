Занятие 9. Структуры. Массив структур. Вариант 2. Определить структурный тип, описывающий гостиничный номер (название гостиницы, номер, комфортность (люкс, полулюкс стандарт, эконом), количество человек, стоимость, бытовые опции). Заполнить структурный массив 20-ю записями. Переписать из исходного массива в другой массив, информацию только о тех гостиничных номерах, название гостиницы которых оканчивается на сочетание букв «plaza». Затем новый массив отсортировать по возрастанию стоимости (рационально переставлять все поля структуры разом). Вывести все данные по конкретному номеру. Вывести 3 самых дешевых номера и их бытовые опции. Реализовать функцию изменения данных конкретного номера. В отдельный массив поместить все номера, имеющие окна на море. Реализовать вывод отфильтрованных данных в виде оберточной функции.

Занятие 10. Чтение и запись данных в файл.
Задание 1. Подпрограмма updateSeaViewFromFile читает из текстового файла myfile.txt пары «название отеля – значение 0 или 1» (аналогично примеру с ФИО и полом) и обновляет соответствующее поле sea_view в структуре room – если название отеля из файла совпадает с названием в массиве, полю sea_view присваивается считанное значение (1 – вид на море есть, 0 – нет). 
Задание 2. Подпрограмма saveRoomsToBinary записывает массив структур room в бинарный файл rooms.bin (сначала количество элементов, затем сами структуры), а подпрограмма loadRoomsFromBinary читает этот файл обратно в массив – таким образом реализовано сохранение и загрузка данных в бинарном виде.
